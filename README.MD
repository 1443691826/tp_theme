**由于造成原因尚不明确，强烈建议用评论过滤插件代替！！**

本主题支持无插件显示表情，可以在评论和发文章时使用：
```
：滑稽：
：喷：
：彩虹：
：阴险：
：怒：
：笑眼：
：黑线：
：玫瑰：
：乖：
：汗：
：惊哭：
：酷：
：狂汗：
：冷：
：真棒：
```

还有-弱-智-无-用-的HTML简化，同样也可以在评论和发文章时使用：```【html标签 你要写的内容 重复html标签】```
比如：
```
【del我是删除线del】会转化成-> <del>我是删除线</del>
【b我我我我b】会转化成-> <b>我我我我</b>
```
*（注意：标签为script的会自动过滤掉，以阻止恶意攻击）*
